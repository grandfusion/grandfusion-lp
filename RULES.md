# GrandFusion サイト制作｜絶対ルール

すべてのページ・すべての作業で必ず守る。例外は「ユーザーからの明示的な指示があったとき」のみ。

## ルール1：ヘッダーは既存HPの構成を厳守（基本、触らない）
- ヘッダーのメニュー項目・並び・ロゴは、**既存の公式サイト（grandfusion-official.site）の構成**に合わせる。
- 勝手に項目を増やす／減らす／並べ替える／英語化する等は**禁止**。
- 変更は **ユーザーの明示指示があった場合のみ**。
- デザイン（色・質感・フォント）は新デザインに調和させてよいが、**構成（項目・並び）は触らない**。
- **全ページでヘッダーは同一**にする（リンクの相対パスのみ各階層に合わせる）。
- 現在の正規ヘッダー項目（＝既存HP準拠＋指示による調整）:
  `エステサロン経営 / エステマシン販売 / ヌドゥーク / 美容室 / BLOG / イベント・セミナー / お問い合わせ`
  - ※「ローマピンク」「商品卸売」「アカデミー運営」はユーザー指示により削除済み（2026-06）。
  - ※「美容室」は新規ページを作らず既存WPページ `https://grandfusion-official.site/hair_salon/` にリンク。

## ルール2：バックアップを取ってから作業
- **既存WordPress（本番）に影響する作業の前に、必ず完全バックアップを取得**しておく。
- バックアップ未取得のうちは、**本番WordPressへの反映・変更は行わない**。
- ローカル（モック）での制作・編集は本番に影響しないため可。ただし**本番反映はバックアップ確認後**に限る。
- バックアップは `.wpress`（All-in-One WP Migration 等）をユーザーが取得し共有する運用。
